# Kalkulator CLI

Proyek kalkulator command-line sederhana menggunakan Python. Proyek ini adalah bagian dari pembelajaran dasar Python untuk memahami input/output dan fungsi dasar.

## Fitur

- Penjumlahan
- Pengurangan
- Perkalian
- Pembagian
- Modulus (sisa pembagian)
- Perpangkatan

## Cara Penggunaan

1. Pastikan Python sudah terinstal di komputer Anda
2. Jalankan program dengan perintah:
   ```
   python calculator.py
   ```
3. Ikuti instruksi yang muncul di layar
4. Masukkan angka dan operator yang diinginkan
5. Lihat hasil perhitungan

## Konsep yang Dipelajari

- Input dan output dasar Python
- Fungsi dan parameter
- Operator aritmetika
- Percabangan (if/else)
- Loop dasar
- Error handling sederhana
